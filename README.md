# Blabber

A mean stack twitter clone.

### Pre-requirements

Install [Nodejs](http://nodejs.org) and [MongoDB](http://www.mongodb.org)

Install grunt and bower globally

```
npm install -g bower grunt-cli
```

### Install

Install local dependencies:

```
npm install

bower install
```

### Running

```
node server.js
```

### Testing

```
grunt test
```
